Shortcut Cheat Sheet

  • Ctrl + Z: Will undo your previous action. (ex. will restore a deleted file, or delete something you just typed on a document.)

  • Ctrl + W: Will close a tab, browser, or open file without having to press the "close" button.

  • Ctrl + A: An EXTREMELY useful shortcut. Allows you to select all text instead of click ing and dragging your mouse over the text.

  • Ctrl + C: Allows you to copy selected/highlighted text and then use Ctrl + V to paste it somewhere else.

  • Ctrl + D: Makes a bookmark.

  • Ctrl + X: Cuts/deletes highlighted/selected text.

  • Ctrl + N: Creates a completely new window.

  • Ctrl + B: Allows you to bold and unbold hightlighted/selected text.

  • Ctrl + I: Allows you to set hightlighted/selected text in italics.

  • Ctrl + Shift + T: Restores a closed tab.

  • Ctrl + Shift + Tab: Switches tabs, going towards the left.

  • Alt + Tab: Switches the app your using, going towards the right.

  • Alt + F4: Closes all apps so you don't have to close them one by one.

    • When holding down the Alt key on a computer, you can type numbers to make certain symbols. (ex. Alt + 0169 = Copyright, Alt + 0174 = Trademark, etc.)
