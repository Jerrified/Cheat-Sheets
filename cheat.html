HTML Cheat Sheet

The Basics

HTML is a programming language used to create and display webpages.
HTML stands for HyperText Markup Language.
You always want to start an HTML document with <!doctype html>. This tells the computer that you are indeed working on an HTML document.
After that, you want to use <html> to reinforce the fact that you are making an HTML document.
You always want a <head>, which contains the <title> and other metadeta.
The <title> is what appears in your tab/web browser.
The <body> is where everything on the webpage actually appears. The <body> contains numerous tags, elements, and attributes.
An element is a single component of an HTML document.
A tag is used to mark the start and end of an element.
Attributes rovide additional information telling the browser what to do to that tag.

Here's how the standard HTML document is built!
Headings are made using the <h1>. There are six levels of headings, from <h1> to <h6>.

<p> is the paragraph element.
<div> allows you to seperate content on an HTML page.
<img> is the image element and requires attributes in order to be completed.
The src (source) attribute points to the location of an image, whether it be the file name or the image address.
The alt attribute provides a description of the image.
Links are added to a webpage using the <a> element, along with the href attribute, which directs the linked text on where to go.
